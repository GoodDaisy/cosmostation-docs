# Introduction

**[Mintscan API](https://api.mintscan.io)** is built on the accumulated data of the Cosmos blockchain over a long period of time.

Our scalable design allows us to provide reliable service.

Based on Mintscan experience using data, we provide an API that can help other builders easily create products needed for the ecosystem.

## Features

- RPC Endpoint
- Historical API
- Proto Builder API
- Statistics API
- Utility API
- Custom API
