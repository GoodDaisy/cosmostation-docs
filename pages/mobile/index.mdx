# Introduction

Mobile wallets support various types of services.

- Access dApps on desktop/mobile via QR code scanning.
- Connect with dApps from your mobile browser via Deep Link.
- In-app browser support on mobile wallets.

---

Cosmostation Mobile Wallet supports dApps through various methods.

- Cosmostation(Same way as Extension)
- [Cosmos-Kit](https://github.com/cosmology-tech/cosmos-kit)
- WalletConnect 1.0(Deprecated), WalletConnect 2.0

**_Connections using Cosmostation and WalletConnect also implement the Keplr and Web3j specifications._**
