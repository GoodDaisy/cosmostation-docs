# Chain Staking APR

## **[GET]** /v1/:network/apr

Get default staking APR of the chain

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: cosmos
```

<TryAPI method="GET" host="http://148.113.165.91" url="/v1/:network/apr" useBearerAuthorization />

## **[GET]** /v1/:network/apr/:validatorAddress

Get staking APR when delegating to a specific validator

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: cosmos
# Validator address to delegate
# (string, required)
validatorAddress: cosmosvaloper1clpqr4nrk4khgkxj78fcwwh6dl3uw4epsluffn
```
