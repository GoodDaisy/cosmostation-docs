# Node Info

## **[GET]** /v1/:network/node_info

Get node informations with chain parameters

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: cosmos
```
