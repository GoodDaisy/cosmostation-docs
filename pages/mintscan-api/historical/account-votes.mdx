# Account Votes

## [GET] /v1/:network/accounts/:address/transactions

Get votes list of account

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: cosmos
# The address of account
# (string, required)
address: cosmos1clpqr4nrk4khgkxj78fcwwh6dl3uw4ep4tgu9q
```

```yaml filename="Queries"
# Whether to remove duplicates for the same proposal
# (boolean, optional, default: false)
distinct: true
```
