# Account Balance History

## **[GET]** /v1/:network/accounts/:address/bank_histories/:symbol

Get balance history of an account

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: cosmos
# The address of account
# (string, required)
address: cosmos1clpqr4nrk4khgkxj78fcwwh6dl3uw4ep4tgu9q
# Symbol of the asset to be queried
# (string, reequired)
symbol: uatom
```

```yaml filename="Queries"
# Reqeust per page
# (number, optional, default: 20)
take: 20
# Value used to skipping a certain number of items
# (string, optional, default: 0)
skip: 0
# Search start datetime
# (datetime, optional, default: 30 days ago)
fromDateTime: 2023-06-20
# Search end datetime
# (datetime, optional, default: CURRENT_TIMESTAMP)
toDateTime: 2023-06-21 23:59:59
```

<TryAPI
  method="GET"
  host="http://3.88.109.66"
  url="/v1/:network/accounts/:address/bank_histories/:symbol"
  query={[
    { key: 'take', type: 'number', optional: true, placeholder: '20' },
    {
      key: 'skip',
      type: 'number',
      optional: true,
      placeholder: '0',
    },
    {
      key: 'fromDateTime',
      type: 'datetime',
      optional: true,
      placeholder: 'YYYY-MM-DD OR YYYY-MM-DD HH:mm:ii',
    },
    {
      key: 'toDateTime',
      type: 'datetime',
      optional: true,
      placeholder: 'YYYY-MM-DD OR YYYY-MM-DD HH:mm:ii',
    },
  ]}
  useBearerAuthorization
/>
